нужно создать сервис такой, чтобы он умел делать две основные вещи:
  1. Пользователь может на сервис передать список дат, а сервис отдать пользователю
     какие из них являются выходными днями (я рассмотривал выходные: сб и вс)
  2. Пользователь может передать массив объектов с ключами date, value на сервис, 
     а сервис возратит те, у которых флаг value = true

для создания API, буду использовать фреймворк - Express: http://expressjs.com/ru/api.html#app

Для решения буду использовать body-parser

откроем порт, пропишем маршруты (роуты), затем напишем пару вспомогательных функций

Декомпозиция:
	1. пишем функцию, которая опр-ет является ли день выходным.  дата - это объект
	2. определить хранилище для дат (массива дат)
	3. логика по обработке входных запросов
	4. логика сохранения и удаления дат
	5. навесить роуты на логику (идея API в том, что при заходе на адрес выполняется логика)
	6. Юнит-тесты
	7. интерфейс
